【注意】
	このゴーストには18歳未満は使用禁止です。

【概要】
	エロRPG世界の 冒険者たちの 雑談

	\0（→側）
	ママ
	あなたのママです。職業は戦士。裸族。
	
	\1（←側）
	ママの友達
	ママの仲間です。職業はアイテム士。裸族。

【さわり反応】
	メニューを出す：マウスカーソルが変化しない場所

	顔見つめ：顔の上でカーソル静止

	胸つつき：胸をクリック
	胸見つめ：胸の上でカーソル静止

	秘部つつき：秘部をクリック
	秘部見つめ：秘部の上でカーソル静止
